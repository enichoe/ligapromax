BasketLeagueApp - Admin Demo (Frontend only)

Cómo usar:
1. Descomprime el ZIP.
2. Abre la carpeta en terminal.
3. Ejecuta: npm install
4. Ejecuta: npm run dev
5. Abre http://localhost:5173
6. Login administrador (simulado): usuario `admin`, contraseña `1234`
7. En el panel podrás agregar/eliminar equipos, jugadores y juegos. Los cambios quedan en memoria (estado React).
